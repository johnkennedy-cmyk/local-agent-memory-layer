## Local-Only Workspace

This folder is for local/internal scripts, notes, and migration helpers that should not be pushed to GitHub.

- Anything under `local-only/` is git-ignored by default.
- Keep production-facing docs and scripts in tracked locations.
- Use this folder for one-off migrations, diagnostics, and local state artifacts.
- `local-only/internal-scripts/` is the recommended place for legacy/internal-only utilities.
